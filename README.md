# Iris Flower Classification using Machine Learning

## Project Overview

This project focuses on classifying Iris flowers into three different species using Machine Learning algorithms. The classification is based on four flower measurements:

* Sepal Length
* Sepal Width
* Petal Length
* Petal Width

The objective is to build, train, and compare multiple classification models to accurately predict the species of an Iris flower.

---

## Dataset Information

The Iris dataset is one of the most widely used datasets for introductory Machine Learning tasks.

### Dataset Features

* Total Samples: 150
* Number of Features: 4
* Number of Classes: 3

### Target Classes

* Iris Setosa
* Iris Versicolor
* Iris Virginica

## 🤖 Machine Learning Models Implemented
